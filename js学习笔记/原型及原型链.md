## 构造函数
```
function Person() {

}
var person = new Person();
person.name = 'Kevin';
console.log(person.name) // Kevin
```
这边简单的构造函数有两个知识点
> 构造函数一般都以大写字母开头；非构造函数是小写开头

> 任何函数以new操作符调用，那他就可以作为构造函数

## prototype, __proto__及constructor的关系
```
以上面构造函数Person为例
每个函数都有prototype属性，它指向实例实例原型
每个对象（null除外），都有__proto__属性，也指向实例原型
实例原型都有constructor
```
